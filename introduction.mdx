## Overview

yashwanth-test2 provides a centralized space for your project documentation. You create, organize, and manage all docs in one place with intuitive tools that support Markdown, MDX, and custom components. Focus on content while it handles hosting, search, and versioning.

This default space comes pre-configured for quick setup, letting you customize themes, add integrations, and collaborate with your team seamlessly.

## Quick Start

Get started in minutes. Follow these steps to set up your first documentation page.

<Steps>
  <Step title="Create Account" icon="user-plus">
    Sign up at `https://dashboard.example.com` with your email.
  </Step>
  <Step title="Initialize Space" icon="settings">
    Create a new documentation space named `yashwanth-test2`.
  </Step>
  <Step title="Add Content" icon="edit-3">
    Upload your first MDX file to `/introduction.mdx`.
  </Step>
  <Step title="Publish" icon="globe">
    Deploy live at `https://docs.example.com`.
  </Step>
</Steps>

## Styling Guidelines

Match your brand using the primary color `#3B82F6`. Apply it consistently for a professional look.

<CodeGroup tabs="CSS, Tailwind">
  ```css
  :root {
    --brand-primary: #3B82F6;
  }

  .button {
    background-color: var(--brand-primary);
    color: white;
  }
  ```
  ```jsx
  // Tailwind config
  module.exports = {
    theme: {
      extend: {
        colors: {
          primary: '#3B82F6',
        },
      },
    },
  };
  ```
</CodeGroup>
