## Prerequisites

Before you begin, ensure you have the following:

<Callout kind="info">

- A modern web browser like Chrome, Firefox, or Safari
- An email address for account verification
- Optional: Node.js `{>=18.0.0}` if using the CLI tools

</Callout>

## Sign Up and Log In

Create your account in under a minute.

<Steps>
  <Step title="Visit the Dashboard" icon="monitor">
    Open your browser and navigate to `https://app.yashwanth-test2.com`.
  </Step>
  <Step title="Register" icon="user-plus">
    Click **Sign Up** and enter your email, password, and name. Verify your email via the confirmation link.
  </Step>
  <Step title="Log In" icon="log-in">
    Return to the dashboard and sign in with your credentials. You'll land on the main workspace overview.
  </Step>
</Steps>

## Create a New Documentation Space

Organize your projects with dedicated spaces.

<Steps>
  <Step title="New Space" icon="plus">
    From the dashboard, click **New Space** in the top-right corner.
  </Step>
  <Step title="Configure Space" icon="settings">
    Enter a name like `My Project Docs`, select visibility (`public` or `private`), and add a description.
  </Step>
  <Step title="Create" icon="check-circle">
    Click **Create Space**. It appears in your workspace list instantly.
  </Step>
</Steps>

## Add Your First Document

Add content via the web UI or API.

<Tabs>
  <Tab title="Web UI" icon="mouse-pointer">

    <Steps>
      <Step title="New Document" icon="file-plus">
        In your space, click **New Document**.
      </Step>
      <Step title="Edit Content" icon="edit-3">
        Use the MDX editor to add markdown. For example:

````mdx
# Welcome

This is your first document in yashwanth-test2.
````

      </Step>
      <Step title="Publish" icon="upload">
        Click **Publish** to make it live.
      </Step>
    </Steps>

  </Tab>
  <Tab title="API" icon="api">

    Use the REST API to create documents programmatically.

    <CodeGroup tabs="JavaScript,cURL">
```javascript
const response = await fetch('https://api.yashwanth-test2.com/v1/spaces/{SPACE_ID}/documents', {
  method: 'POST',
  headers: {
    'Authorization': 'Bearer YOUR_API_KEY',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    title: 'API Document',
    content: '# Created via API\nThis uses the yashwanth-test2 API.'
  })
});
```
```bash
curl -X POST https://api.yashwanth-test2.com/v1/spaces/{SPACE_ID}/documents \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "API Document",
    "content": "# Created via API\nThis uses the yashwanth-test2 API."
  }'
```
    </CodeGroup>

    Replace `{SPACE_ID}` with your space ID and `YOUR_API_KEY` from dashboard settings.

  </Tab>
</Tabs>

<ParamField path="spaceId" param-type="string" required="true">
  Unique space identifier from the dashboard URL.
</ParamField>

<ParamField header="Authorization" param-type="string" required="true">
  Bearer token: `Bearer YOUR_API_KEY`.
</ParamField>
