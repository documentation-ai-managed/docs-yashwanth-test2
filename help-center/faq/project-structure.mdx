## Overview

yashwanth-test2 organizes docs into sections and pages.

## Default Structure

- **Home**: Project overview
- **Sections**: Group related pages (Guides, API, etc.)
- **Pages**: Each doc lives in a section
- **Sidebar**: Navigate your docs easily

## How to Customize

- Add, rename, or reorder sections
- Drag pages between sections
- Use sidebar for quick access
